# Bidcms商城小程序

#### 项目介绍
Bidcms开源小程序是由Bidcms开源电商开发的一套快捷生成小程序的开源程序，仅作为研究交流使用
无需安装 ，直接运行即可。此版本可满足一般小程序发展下级代理，二次开发，无任何限制，
包含如下功能：
1、自带26套模板
2、微信商城，满足顾客多场景自主购物需求
3、完整的零售管理后台
4、全渠道支撑 ，微商城、小程序、门店

#### 安装教程

1、上传到您自己的服务器根目录
2、直接运行index.php

1. 添加商铺，添加后操作
2. 如需设置自己的域名，点击设置->域名管理->等待审核
